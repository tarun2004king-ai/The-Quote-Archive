#  The Quote Archive

A **MERN Stack Quote Generator** that fetches random inspirational quotes from the **ZenQuotes API** and stores every viewed quote in **MongoDB**, presented through a clean vintage library card interface.

##  Functionality

-  Fetches a random quote from the ZenQuotes API
-  Automatically saves each viewed quote to MongoDB
-  Displays recently viewed quotes in chronological order
-  Delete individual quotes from history
-  Clear the entire quote history
-  Uses local fallback quotes if the external API is unavailable
-  Responsive vintage-inspired UI built with React

##  Tech Stack

- **Frontend:** React, Vite, CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB, Mongoose
- **External API:** ZenQuotes API
